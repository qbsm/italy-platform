# Доступность (a11y)

## Целевой уровень WCAG

Проект ориентируется на **WCAG 2.1 Level AA** для публичных страниц: контраст, навигация с клавиатуры, тексты для изображений, структура заголовков, формы с подписями и сообщениями об ошибках.

## Реализовано

- **Skip-link** — ссылка «Перейти к контенту» в начале страницы (`templates/base.twig`), скрыта до получения фокуса (клавиатура или Tab). Цель: `#page-content` (`templates/pages/page.twig`).
- **Фокус** — при необходимости управление фокусом в модалках и меню описывается в компонентах (burger-menu, модальные окна).

## Рекомендации

- Проверять порядок фокуса при открытии/закрытии модалок и выпадающих меню.
- Для интерактивных элементов без видимого текста использовать `aria-label` или скрытый текст.
- Поддерживать контраст текста и фона не ниже 4.5:1 (обычный текст), 3:1 (крупный).
